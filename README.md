## Refine the OpenCV mask

__Dataset:__ Satellite Images of different areas in delhi

__Requirements:__ 
  + Python 3
  + cv2 (OpenCV)
  + numpy
 
__Concepts used:__
  * Thresholding
  * Morphology
  * Distance Transform
  * Watershed algorithm
